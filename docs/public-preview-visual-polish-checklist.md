# Drishvara Public Preview Visual Polish Checklist

## Purpose

B25 checks the visible public-preview experience after the architecture, placeholder guards, timezone basis, and scaffold safety layers are in place.

## Homepage

- Top navigation remains compact.
- IST dropdown appears in the nav and does not disturb the homepage grid.
- No large date-basis card appears in the body.
- Sports live object is visible but not visually dominant.
- Featured Reads retain images and readable card layout.
- Hindi/English toggle remains usable.
- Public modules do not claim fake live or unreviewed Panchang/Vedic output.

## Article Reader

- Title size is readable.
- Article width is comfortable.
- Paragraph text remains readable and justified where configured.
- Featured image is visible and not oversized.
- Hindi body switching remains intact.

## Submissions Page

- Backend intake disabled status remains visible.
- Palm image upload remains disabled.
- Prepared packet / copy / email flow remains understandable.

## Dashboard

- Login/subscription pending state is clearly visible.
- Lucky number, lucky color, mantra, and reflection remain blocked.
- No profile data collection is active.

## Admin

- Admin actions disabled state is visible.
- Page remains noindex.
- No live backend actions are present.

## Mobile

- Navigation wraps cleanly.
- IST dropdown does not overflow.
- Sports pill becomes page-flow item on smaller screens.
