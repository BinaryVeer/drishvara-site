# Drishvara Knowledge Stage D01 — Daily Guidance & Panchang Engine Governance

## Purpose

D01 creates the governance foundation for future dynamic Daily Guidance, Panchang, Festival, Mantra, Word of the Day, and reflective spiritual guidance modules.

This stage is deliberately governance-only.

## Current Public Position

The current public site may show static/scaffold daily modules. These should not be treated as verified dynamic Panchang or personalized spiritual output.

## Future Engine Areas

- Word of the Day rotation
- Vedic Guidance logic
- Panchang and Festival view
- Mantra selection
- Lucky colour and number
- Subscriber daily guidance

## Strict Guardrails

This stage does not:

- change language runtime
- translate article bodies
- fetch external APIs
- enable Supabase
- enable Auth
- enable payment
- unlock premium guidance
- enable admin actions
- claim Panchang/Festival/Mantra output is dynamic
- mutate generated article files

## Review Requirement

All Panchang, Sanskrit, mantra, tithi, nakshatra, festival, vrata, deity, and spiritual guidance rules must be reviewed before public dynamic use.
