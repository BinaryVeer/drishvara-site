# Knowledge Stage D05 — Panchang/Festival Source Registry & Validation Preview

## Purpose

D05 creates a source registry and validation preview structure for future Panchang and Festival features.

This stage is not live. It does not calculate Panchang values and does not assign festival dates.

## Current Public Position

The public Panchang/Festival display remains scaffolded or blocked. D05 does not update `data/daily-context.json`.

## Files Added

- `data/knowledge/sanatan/panchang-festival-source-registry-d05.json`
- `data/knowledge/sanatan/festival-observance-registry-d05.json`
- `data/knowledge/sanatan/panchang-festival-validation-preview-d05.json`

## Source Rules

Future Panchang/Festival data must be validated through recognized references, reviewed calculation logic, and manual editorial approval before public use.

## This Stage Does Not

This stage does not fetch external APIs, does not calculate live Panchang values, does not assign festival dates, does not change language runtime, does not enable Supabase/Auth/payment/admin features, does not unlock premium guidance, and does not write into public daily context.

## Review Rule

Any future tithi, nakshatra, festival, observance, sunrise, sunset, yoga, karana, mantra, vrata, or spiritual guidance output must be reviewed before public dynamic use.
