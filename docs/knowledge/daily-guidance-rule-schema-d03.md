# Knowledge Stage D03 — Daily Guidance Rule Schema

## Purpose

D03 defines the future rule structure for Drishvara Daily Guidance.

This is schema-only. It does not make public guidance dynamic and does not unlock subscriber or premium guidance.

## Rule Areas

Future rules may be based on:

- weekday
- tithi
- nakshatra
- season
- festival or observance
- Word of the Day theme
- neutral fallback

## Output Style

Daily guidance should remain reflective, non-deterministic, and non-alarmist. It should support thoughtful conduct rather than making certainty-based predictions.

## This Stage Does Not

This stage does not change the language runtime, does not fetch external APIs, does not enable Supabase/Auth/payment/admin features, does not unlock premium guidance, and does not make Panchang/Vedic guidance dynamic.

## Review Requirement

All daily guidance rules must be reviewed before public use, especially rules involving Sanskrit, mantra, deity, vrata, tithi, nakshatra, festival, or spiritual interpretation.
