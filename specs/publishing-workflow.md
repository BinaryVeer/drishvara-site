# Drishvara Publishing Workflow

## 1. Topic generation
Generate 8–12 candidates per category:
- spirituality
- sports
- world_affairs
- media_society
- public_programmes

## 2. Topic scoring
Score each candidate on:
- Buzz
- Intent clarity
- Public relevance
- Drishvara fit
- Research depth

## 3. Research packet
Build structured packet with:
- angle
- why now
- facts
- timeline
- references
- image note
- risk note

## 4. Writing
Write in Drishvara tone:
- calm
- reflective
- precise
- not clickbait
- signal over spectacle

## 5. Publish
Create article file in:
- articles/spiritual/
- articles/sports/
- articles/world/
- articles/media/
- articles/policy/

## 6. Link update
Update:
- homepage sample card if needed
- insights card for that category

## 7. Signal rail
Use shortlisted topics not selected for the Daily Signal Rail.

## 8. Star reflection
User enters:
- Name
- DOB
- language

Return:
- theme
- focus
- caution
- opportunity
- reflection

## 9. Language
Phase 1:
- English UI
- Hindi UI option
- English articles
- Hindi star reflections

Phase 2:
- bilingual articles
